# Leaf Logistics

A simple REST api for transposing matrices.

## Dependencies

- Python
- Pip
- Flask

## Setup

- clone repo
- python3 -m venv venv
- source venv/bin/activate
- flask run

## Test API

`curl -d '[[1,2,3],[4,5,6],[7,8,9]] ' -H 'Content-Type: application/json' http://127.0.0.1:5000/`
